# AI Automation Workflow (In Progress)

This project is a prototype for an AI-driven recruitment automation system that automates candidate screening using AI and structured workflows.

## 🚀 Workflow Overview
1. Input: Candidate resume data
2. Processing: AI-based evaluation (LLM logic)
3. Output: Structured JSON shortlist
4. Feedback Loop: HR feedback used to refine results

## 🧠 Key Features
- Simulates AI-powered candidate evaluation
- Generates structured JSON output
- Designed for integration with n8n workflows
- Supports iterative improvement using feedback loops

## 🛠️ Tech Stack (Planned)
- n8n (Workflow Automation)
- Gemini API (LLM Integration)
- JSON for structured data handling

## 📌 Status
Currently under development. Initial workflow logic defined and implementation in progress.

## 📈 Future Improvements
- Full n8n workflow integration
- Live API calls to Gemini
