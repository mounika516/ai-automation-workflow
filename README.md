\# AI Automation Workflow



An AI-driven candidate screening system.



\## Tech Stack

\- Python 3.14

\- Google Gemini API

\- Scikit-learn and Pandas

\- n8n workflow automation



\## Setup

1\. Clone the repo

2\. Run: pip install -r requirements.txt

3\. Add GEMINI\_API\_KEY to .env file

4\. Run the pipeline
