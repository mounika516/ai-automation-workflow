# ai-automation-workflow
This project is a prototype for an AI-driven recruitment automation system.

## Workflow Idea:
1. Input: Candidate resume data
2. Processing: AI-based evaluation
3. Output: Structured JSON shortlist
4. Feedback Loop: HR input improves results

## Status:
Currently under development. Will integrate n8n workflows and Gemini API.

## Tech Focus:
- Automation workflows (n8n)
- AI API integration (Gemini)
- JSON structured output
